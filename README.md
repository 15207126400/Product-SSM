# Product-SSM
基于原生SSM框架编写的商城管理系统
# 项目介绍
核心框架: Spring</br>
持久层框架: MyBatis</br>
交互层框架: SpringMVC</br>
数据库: MySql</br>
缓存: Redis</br>
数据库连接池: Alibaba Druid Pool</br>
文件上传: Ftp</br>
富文本插件: Ueditor</br>
日志管理: Log4j</br>
模板引擎: Freemark+Bootstrap+EasyUi</br>
